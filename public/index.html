🔌 REST API Reference

    
All endpoints accept and return JSON. The /convert endpoint returns a PDF binary.


    
    

      

        GET
        /health
        Service health check
      

      

        

          
Request

          
curl https://your-service.run.app/health

        

        

          
Response 200 OK

          
{
  "status": "ok",
  "service": "md2pdf-lab",
  "version": "1.0.0",
  "uptime": 42
}

        

        
        

      

    


    
    

      

        POST
        /convert
        Convert Markdown to PDF
      

      

        

          
FieldTypeDefaultDescription

          
            
markdownstringrequiredRaw Markdown content

            
themelight|dark|githublightPDF visual theme

            
pageSizeA4|Letter|LegalA4Page dimensions

            
marginnumber (0-50)15Margin in mm

            
tocbooleantrueTable of contents

            
mathbooleantrueMathJax rendering

            
mermaidbooleantrueMermaid diagrams

            
displayHeaderFooterbooleanfalsePage header/footer

            
filenamestringdocumentDownload filename

          
        

        

          
Request

          
curl -X POST /convert \
  -H "Content-Type: application/json" \
  -d '{"markdown":"# Hello","theme":"dark"}' \
  --output doc.pdf

        

        

          
Response

          
200 OK
Content-Type: application/pdf
Content-Disposition: attachment; filename="Hello.pdf"

[binary PDF data]

        

      

    


    
    

      

        GET
        /preview?md=...&theme=dark
        Render Markdown as HTML in browser
      

      

        

          
Example

          
curl "/preview?md=%23%20Hello&theme=dark"
